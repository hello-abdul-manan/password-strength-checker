# üîê Password Strength Checker

A simple Python CLI tool that checks if a password is strong based on basic rules:
- At least 8 characters
- Contains uppercase
- Contains lowercase
- Contains a digit

## Features
- Evaluates strength of a password
- Rejects weak passwords
